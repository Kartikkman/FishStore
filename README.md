# FishStore
A simple React app demonstrating the fish store 

Based on Wes Bos' ReactForBeginners.com.
Worked on this app to learn basic concecpts of React ( my first React Project).

### [Live](https://fish-store-react.netlify.app/)

## Functionality 
-  User opens up the App  & can create his own store ( where he will be able to sell fishes )  . 
- Ability to add fishe with price ( can also load sample fishes ) 
- Add fish to Cart ( creates totatl ) 
- CRUD operations to Add , Update or Delete the Fish
